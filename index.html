Blog Formatter

Paste once — get formatted WordPress blocks + excerpt

How to Use

  1. Copy the entire blog content from your Word document (Ctrl+A, Ctrl+C)
  2. Paste it in the box below — bold & italic formatting is preserved
  3. Click Format for WordPress
  4. In WordPress: Click the ⋮ menu (top right) → Code editor
  5. Select all existing code (Ctrl+A) and paste the generated code (Ctrl+V)
  6. Click ⋮ menu → Visual editor to return to normal view
  7. Add your images to the placeholder spots
✓ Bold and italic text from Word will be preserved when you paste
